속성명이 tit(특정 문자)인 경우 스타일 설정
[ tit ]{tit속성이 있는 전부 해당}
속성명과 속성값이 같은 경우
[ tit = "tt p" ]{3번 해당}
속성값이 해당 이름을 포함하고 있는 경우
[ tit ~= "tt" ]{1,2,3,5번 해당}
속성값이 해당 이름으로 시작되는 단어만으로 된경우 (-는 연결로 인정)
[ tit |= "tt" ]{2,4번 해당}
속성값이 해당값으로 시작하는 문자열인 경우
[ tit ^= "tt" ]{1,2,3,4번 해당}
속성값이 해당값으로 문자열이 끝나는 경우
[ tit $= "tt" ]{2,5,6번 해당}
속성값이 해당 문자열을 포함한 경우
[ tit *= "tt" ]{1~7번 해당}
